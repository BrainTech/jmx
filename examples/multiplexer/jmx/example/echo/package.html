This is an example of simple Java Multiplexer installation. Configuration of the {@link multiplexer.jmx.server.JmxServer} is statically embedded in the code to simplify configuration even further (see {@link multiplexer.jmx.example.echo.JmxServerRunner}).

To run this example run {@code main} static methods of {@link multiplexer.jmx.example.echo.JmxServerRunner}, {@link multiplexer.jmx.example.echo.EchoBackend} and then {@link multiplexer.jmx.example.echo.EchoClient}. To ensure that all requires classes are in the classpath run {@code ant jar} and then use java's option: {@code -cp bin:dist/jmx-*-withdeps.jar}.
